# Parking Meter System (CLI)

A simple command-line based parking meter system that calculates parking fees 
based on check-in and check-out time and processes cash payments.

## Features
- Calculates parking fees based on time range
- Handles invalid input (e.g., time > 24 hours)
- Accepts multiple payments until full amount is paid
- Returns change if excess amount is entered

## Concepts Used
- Loops
- Conditional statements
- Input validation

## Note
This project was built under constraints:
- No functions
- No classes
- No lists
- No external libraries
- NO OOP
